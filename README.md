# Expense Tracker Analysis 💰

## Objective
To analyze personal spending habits and identify patterns in expenses.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Total spending by category
- Daily spending trends
- Category distribution
- Average spending by category

## Insights
- Major expenses are concentrated in specific categories like Bills and Shopping
- Daily expenses show consistent spending patterns
- Some categories have higher average spending
- Opportunities exist to reduce unnecessary expenses

## Conclusion
This project helps in understanding spending behavior and making better financial decisions.
